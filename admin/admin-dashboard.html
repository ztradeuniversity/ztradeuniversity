About Unmatched Accounts

Some accounts may show Waiting for Match or Match Not Found. This usually happens because:

  • The broker referral report has not been updated yet — this can take a few days
  • The referral link or IB code was not applied correctly during sign-up
  • The client used an existing broker account instead of creating a new one
  • The app redirected incorrectly during the registration flow

These requests will be reviewed automatically when a new broker report is uploaded. No action is needed right now.
