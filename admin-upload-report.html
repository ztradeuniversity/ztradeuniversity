How processing works

  1. 1
    Parse the broker file (CSV via PapaParse, XLSX via SheetJS).
  2. 2
    Auto-detect the account-number column from common header names.
  3. 3
    Fetch all pending license requests from Supabase.
  4. 4
    Match requests against report accounts and auto-approve the verified ones.
  5. 5
    Queue approved accounts for compilation as ZTU-BOT-{account}.ex5.
Recognized account-column names: account, account number, login, mt5 account, trading account.
